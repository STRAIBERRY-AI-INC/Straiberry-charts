## Getting Started
### Adding dependecies
1. Add it in your root build.gradle at the end of repositories:
  ```sh
  	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  ```
2. Add the dependency
  ```sh
  	dependencies {
	        implementation 'com.github.STRAIBERRY-AI-INC:Straiberry-charts:1.1.0'
	}
  ```
